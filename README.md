# Shoutbox

Shoutbox is a decentralised peer-to-peer chat. It can not be manipulated by any individual or government. Users are able to run their own in-node browser and safely send messages with anyone, anywhere. 

## Tech Stack
Typescript, LibP2P 

## How to Run
npm install webpack

Yarn Start
